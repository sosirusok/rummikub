# 바닐라 마인크래프트 + 하이픽셀 스카이블럭 구현 레퍼런스

> deep-research(95 에이전트, 소스 15개: minecraft.wiki 다수 + hypixel-skyblock.fandom.com) 검증 결과.
> V12~V16 구현 시 이 수치를 그대로 사용한다. 추측 금지.

## 1. 바닐라 인벤토리 구조 (V12)
- **총 41슬롯** = 메인 저장 27 + 핫바 9 + 방어구 4(투구/흉갑/레깅스/부츠) + 오프핸드 1
- UI는 **9×4 = 36칸**(저장 27 + 핫바 9) 그리드 + 우측 방어구 4칸 세로 + 오프핸드 1칸
- 오프핸드는 자바에서 **F키**로 스왑
- 스택 규칙: **대부분 64**, 일부 발사체 16, **내구도 있는 도구/무기/방어구는 스택 불가(1)**
- 핫바 9칸: 플레이어가 원하는 아이템을 원하는 슬롯에 배치(숫자키 1~9 선택)

## 2. 바닐라 초반 조합 (V12) — 정확한 배치
- 원목 1 → **판자 4** (무형 조합)
- 판자 2(세로) → **막대 4**
- 판자 4(2×2) → **작업대 1**
- **곡괭이**: 상단 행 재료 3 + 가운데 열 막대 2 (T자)
- **검**: 가운데 열 재료 2(세로) + 막대 1(아래)
- **도끼**: 재료 3(ㄱ자) + 막대 2
- **삽**: 재료 1 + 막대 2(세로)
- **괭이**: 상단 재료 2 + 막대 2
- **화로**: 조약돌 8 (3×3 외곽, 중앙 빈칸)
- **상자**: 판자 8 (3×3 외곽, 중앙 빈칸)
- 화로: 1아이템 = 200틱 = **10초**, 석탄 1개 = 80초 = 8아이템 제련

## 3. 도구 티어 수치 (V12) — 채굴 속도 / 내구도 / 채굴 레벨 / 공격 보너스
| 티어 | 채굴속도 | 내구도 | 채굴레벨 | 공격보너스 |
|---|---|---|---|---|
| 나무(wood) | 2 | 59 | 0 | 0 |
| 돌(stone) | 4 | 131 | 1 | +1 |
| 철(iron) | 6 | 250 | 2 | +2 |
| 다이아(diamond) | 8 | 1561 | 3 | +3 |
| 네더라이트(netherite) | 9 | 2031 | 4 | +4 |
| 금(gold) | 12(가장 빠름) | 32(가장 약함) | 0 | - |
- 나무 검: 공격 4HP(하트 2), 공격속도 1.6 → DPS 6.4
- **채굴 레벨이 블럭 채취 가능 여부를 게이트**(나무<돌<철<다이아<네더라이트)
- ★ **본 게임 규칙**: 바닐라 도구(다이아·네더라이트 포함) = 여기선 **최하 등급(common)** 취급. 스카이블럭 커스텀 장비가 훨씬 강함.

## 4. Shift(sneak) 동작 (V12)
- **가장자리에서 안 떨어짐**: 높이차 0.6블럭 초과 낙하 지점에서 걷기 차단
- 이동 속도 **정상 걷기의 30%** (걷기 4.317 b/s → 슬금 ~1.3 b/s, 대각선 1.8)
- 히트박스 높이 1.8 → **1.5블럭**
- 상호작용 블럭 위에 **아이템(블럭/횃불) 배치 우선**(GUI 안 열림)
- 적대 몹 감지 범위 **80%로 감소**

## 5. Hypixel 프라이빗 아일랜드 (V13)
- 신규 시작 시 **티어 I 조약돌 미니언 1개 이미 배치**된 상태
- **미니언은 섬 어디에나 자유 배치**(고정 위치 아님), 5×5 범위에서 생산
- 기본 **미니언 슬롯 5개**, 고유 미니언 조합 + 커뮤니티 업그레이드로 **최대 31**(조합 26 + 커뮤니티 5)
- 미니언 슬롯 4종: 스킨(x1) / 연료(x1, 생산속도↑) / 자동배송(x1, 가득 차면 자동판매) / 업그레이드(x2)
- 미니언 티어 I~XI(일부 XII)
- **프라이빗 아일랜드 → 허브가 첫 이동 섬**(SkyBlock 메뉴 워프 또는 포탈)

## 6. Hypixel 허브 구역 (V14)
- 허브는 프라이빗 섬을 떠나 처음 가는 **중앙 섬**, 중앙에 **Village**, 거기서 스킬별 전문 구역으로 분기
- **Village 내부 4개 명소**:
  - **Auction House**(경매장): 아이템 매매/입찰. NPC Auction Agent, Auction Master, Damia, Shen's Agent
  - **Bank**(은행): 코인 입출금, 개인 금고. NPC Banker
  - **Bazaar Alley**: 바자(즉시 매매). NPC Bazaar, Bazaar Agent
  - **Builder's House**: **건축 블럭 구매**. NPC **Builder**, Mad Redstone Engineer, Wool Weaver
- Village 좌측 광장: 분수 + 던전 허브 포탈, 이벤트 개최
- 스킬 구역: Farm, Coal Mine, Graveyard, Wilderness, Colosseum, Community Center 등
- Hub에 페어리 소울 79개, 워프 스크롤(Hub Castle, Dark Auction, Hub Crypts, Museum, Wizard Tower)

## 7. Hypixel 체력/방어 UI (V15)

## 8. 스카이블럭 진행 수치(V16 딥리서치 검증 — 108에이전트, wiki.hypixel.net 1차)
- **스킬 XP 표(전 스킬 공용)**: L1=50,L2=125,L3=200,L4=300,L5=500,L6=750,L7=1000,L8=1500,L9=2000,L10=3500 … 누적 **L50=55,172,425 / L60=111,672,425**. 49→50=4,000,000, 59→60=7,000,000. (본 게임 SKILL_XP_TABLE과 L50까지 100% 일치, L60 확장 완료)
- **스킬 상한**: 전투/농사/채광/마법부여/조련 60, 낚시 50, 벌목 50(위키 54=2025패치), 사교 25
- **스킬별 레벨 보너스**: 전투=크확 +0.5%/레벨(+워리어 +4~210% 몹피해), 농사=체력, 채광=방어, 벌목=힘, 낚시=체력, 마법부여=지력
- **컬렉션(실제 임계값)**: 조약돌 10티어 [50,100,250,1000,2500,5000,10000,25000,40000,70000], 석탄 [50,100,250,1000,2500,5000,10000,25000,50000,100000]. 각 티어 +4 SB XP + 레시피/스탯 해금. 채집·미니언만 카운트(경매/바자 제외).
- **미니언(V16-B 대상)**: 조약돌 T-I = 14초/행동, 저장 64. 슬롯 5기본→최대 31(고유제작 5=+1,15=+2,50=+4,100=+6,200=+10,350=+15,650=+21 + 상점 +5). 연료: 석탄+5%/30분, 인챈석탄+10%/24h, 인챈용암양동이+25%(영구), 플라즈마+35%, 촉매 3배산출.
- **슬레이어(V16-B 대상)**: 슬레이어레벨 누적XP [5,15,200,1000,5000,20000,100000,400000,1000000](본 게임 일치). 리븐넌트 비용 2000/7500/20000/50000/100000, 보스HP 500/20000/400000/1500000/10000000, 게이지 전투XP 150/1440/2400/4800/6000. 스벤 4티어(T5 없음) HP 2000/40000/750000/2000000.
- **던전 점수**: S+≥300, S≥270, A≥230, B≥160, C≥100. 카타콤/클래스 50레벨(총 XP≈569.8M). 장기 그라인드=시크릿·RNG 장비·재조합기.
- **총 그라인드**: 6스킬 만렙 ≈ 300M+ XP, 컬렉션 1개 만렙 수만 개, 슬레이어9 100만 XP → 수백~수만 시간(다중 곱셈 벽).
